\begin{frame}{シェルとは 2/3}
  \begin{itemize}
    \item カーネル
      \begin{itemize}
        \item OSの核となる（中心的な）機能の実装
          \begin{itemize}
            \item ファイルシステム、ネットワーク、入出力、プロセス管理など
          \end{itemize}
        \item Linuxカーネル、macOSカーネル（XNU）、Windowsカーネル
        \item Linuxディストリビューションと呼ばれるOSのカーネルはLinuxカーネルをベースにしている
          \begin{itemize}
            \item Debian GNU/Linux、Ubuntu、CentOSなど
          \end{itemize}
      \end{itemize}
    \item シェル
      \begin{itemize}
        \item コマンドによりカーネルの機能を利用するアプリケーションの一種
        \item ユーザに提供されるコマンドベースのカーネルへのインタフェース
          \begin{itemize}
            \item カーネル/核 の 殻/シェル
          \end{itemize}
      \end{itemize}
  \end{itemize}

\end{frame}
